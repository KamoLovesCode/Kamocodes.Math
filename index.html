Part 1: Theorems with Proofs and Memory Devices

1. Extreme Value Theorem

Statement: If f is continuous on [a,b], then f attains absolute max and min values on [a,b].

Proof Sketch:

1 Continuity on closed interval → function is bounded
2 Let M = sup{f(x) : x ∈ [a,b]} and m = inf{f(x) : x ∈ [a,b]}
3 By completeness of ℝ, ∃ c,d ∈ [a,b] such that f(c) = M, f(d) = m
Memory Device: "Closed + Continuous = Guaranteed Extremes"

2. Rolle's Theorem

Statement: If f continuous on [a,b], differentiable on (a,b), and f(a) = f(b), then ∃ c ∈ (a,b) with f'(c) = 0.

Proof Sketch:

1 If f constant → f'(x) = 0 everywhere
2 If f not constant → by EVT has max/min in (a,b)
3 By Fermat's Theorem, f'(c) = 0 at that point
Memory Device: "Same height at ends → somewhere flat in between"

3. Mean Value Theorem (MVT)

Statement: If f continuous on [a,b] and differentiable on (a,b), then ∃ c ∈ (a,b) such that:

$$f'(c) = \frac{f(b) - f(a)}{b - a}$$

Proof Sketch:

1 Define helper function: \(g(x) = f(x) - f(a) - \frac{f(b)-f(a)}{b-a}(x-a)\)
2 Check g(a) = g(b) = 0
3 Apply Rolle's Theorem to g(x)
4 g'(c) = 0 → \(f'(c) = \frac{f(b)-f(a)}{b-a}\)
Memory Device: "Some instantaneous rate equals average rate"

4. Fundamental Theorem of Calculus

Part 1: \(\frac{d}{dx}\left[\int_{a}^{x} f(t)dt\right] = f(x)\)

Proof Sketch:

1 Let \(F(x) = \int_{a}^{x} f(t)dt\)
2 \(\frac{F(x+h) - F(x)}{h} \approx \frac{1}{h}\int_{x}^{x+h} f(t)dt \approx f(x)\)
3 Limit h→0 gives F'(x) = f(x)

Part 2: \(\int_{a}^{b} f(x)dx = F(b) - F(a)\) where F' = f

Proof Sketch:

1 Let \(F(x) = \int_{a}^{x} f(t)dt + C\)
2 F(b) - F(a) = \(\int_{a}^{b} f(t)dt\)
Memory Device: "Derivative undoes integral, integral undoes derivative"

Fermat's Theorem - Step-by-Step Proof

Fermat's Theorem (Critical Point Theorem)

Statement: If a function f has a local maximum or minimum at point c, and if f'(c) exists, then f'(c) = 0.

Step-by-Step Proof (Easy to Memorize):

Step 1: Assume Local Maximum
Assume f has a local maximum at c (proof for minimum is similar)
Step 2: Analyze Right-Hand Derivative
For small h > 0: f(c+h) ≤ f(c) (by definition of local max)
Difference quotient: [f(c+h) - f(c)]/h ≤ 0 (numerator ≤ 0, denominator > 0)
Take limit as h→0⁺: f'(c⁺) ≤ 0
Step 3: Analyze Left-Hand Derivative
For small h < 0: f(c+h) ≤ f(c) (by definition of local max)
Difference quotient: [f(c+h) - f(c)]/h ≥ 0 (numerator ≤ 0, denominator < 0)
Take limit as h→0⁻: f'(c⁻) ≥ 0
Step 4: Combine Results
Since f'(c) exists: f'(c⁺) = f'(c⁻) = f'(c)
From Step 2: f'(c) ≤ 0
From Step 3: f'(c) ≥ 0
Only number satisfying both: f'(c) = 0
Step 5: Conclusion
Therefore, if f has local extremum at c and f'(c) exists, then f'(c) = 0
Memory Aid: "At the top of a hill or bottom of a valley, the ground is flat."

Why This Matters for Your Exam:

1 Essential for finding critical points in optimization problems
2 Foundation for curve sketching to locate local extrema
3 Key component in proving other theorems like Rolle's Theorem
4 You might be asked to state, apply, or even reproduce this proof

Mean Value Theorem for Integrals - Step-by-Step Proof

Mean Value Theorem for Integrals

Statement: If f is continuous on [a,b], then there exists some c in [a,b] such that:

$$f(c) = \frac{1}{b-a} \int_{a}^{b} f(x)dx$$

Step-by-Step Proof (Easy to Memorize):

Step 1: Apply Extreme Value Theorem
Since f is continuous on [a,b], by EVT:
f attains absolute minimum m and maximum M on [a,b]
So: m ≤ f(x) ≤ M for all x in [a,b]
Step 2: Apply Integral Properties
Integrate the inequality: ∫ₐᵇ m dx ≤ ∫ₐᵇ f(x)dx ≤ ∫ₐᵇ M dx
Since m and M are constants: m(b-a) ≤ ∫ₐᵇ f(x)dx ≤ M(b-a)
Step 3: Divide by (b-a)
Divide all parts by (b-a): m ≤ [1/(b-a)]∫ₐᵇ f(x)dx ≤ M
Let A = [1/(b-a)]∫ₐᵇ f(x)dx (this is the average value)
So: m ≤ A ≤ M
Step 4: Apply Intermediate Value Theorem
Since f is continuous on [a,b] and A is between m and M
By IVT, there exists c in [a,b] such that f(c) = A
Step 5: Conclusion
Therefore: f(c) = [1/(b-a)]∫ₐᵇ f(x)dx for some c in [a,b]
Memory Aid: "The average height equals some actual height."

Why This Matters for Your Exam:

1 Essential for understanding the concept of average value
2 Applications in physics and engineering
3 Connects integration with function behavior
4 You might be asked to state, apply, or even reproduce this proof

Part 2: Comprehensive Derivatives

Basic Rules

Rule Formula
Power Rule \(\frac{d}{dx}(x^n) = nx^{n-1}\)
Constant Multiple \(\frac{d}{dx}[cf(x)] = cf'(x)\)
Sum/Difference \(\frac{d}{dx}[f(x) \pm g(x)] = f'(x) \pm g'(x)\)
Product Rule \(\frac{d}{dx}[f(x)g(x)] = f'(x)g(x) + f(x)g'(x)\)
Quotient Rule \(\frac{d}{dx}\left[\frac{f(x)}{g(x)}\right] = \frac{f'(x)g(x) - f(x)g'(x)}{[g(x)]^2}\)
Chain Rule \(\frac{d}{dx}[f(g(x))] = f'(g(x))g'(x)\)
