1. Fundamental Theorems (High Priority)

Mean Value Theorem (MVT)

Statement: If f is continuous on [a,b] and differentiable on (a,b), then there exists c ∈ (a,b) such that:

f'(c) = [f(b) - f(a)] / (b - a)

When to use: When you need to prove existence of a point where instantaneous rate equals average rate.

Proof Strategy (MVT):

1. Define g(x) = f(x) - f(a) - [(f(b)-f(a))/(b-a)]·(x-a)
2. Show g(a) = g(b) = 0
3. Apply Rolle's Theorem to g(x)
4. Conclude g'(c) = 0 for some c ∈ (a,b)
5. Show this implies f'(c) = [f(b)-f(a)]/(b-a)

Rolle's Theorem

Statement: If f is continuous on [a,b], differentiable on (a,b), and f(a) = f(b), then there exists c ∈ (a,b) such that f'(c) = 0.

Special Case: Rolle's Theorem is actually a special case of MVT where f(a) = f(b).

Exam Tip:

You MUST be able to state these theorems perfectly and reproduce their proofs. This is typically 4-8 marks guaranteed if you memorize them.

2. Integration Techniques (High Priority)

Integration by Parts

∫ u dv = uv - ∫ v du

LIATE Rule: Choose u in this order: Logarithmic, Inverse trig, Algebraic, Trigonometric, Exponential

Example: ∫ e^x cos(x) dx

Let u = e^x, dv = cos(x) dx
Then du = e^x dx, v = sin(x)
∫ e^x cos(x) dx = e^x sin(x) - ∫ e^x sin(x) dx
Apply integration by parts again to ∫ e^x sin(x) dx
Final answer: ½ e^x (sin(x) + cos(x)) + C

Trigonometric Substitution

√(a² - x²) → use x = a sinθ
√(a² + x²) → use x = a tanθ
√(x² - a²) → use x = a secθ

Partial Fractions

For rational functions P(x)/Q(x) where deg(P) < deg(Q)

Example: ∫ (3x+11)/(x²-x-6) dx

Factor denominator: x²-x-6 = (x-3)(x+2)
Set up: (3x+11)/[(x-3)(x+2)] = A/(x-3) + B/(x+2)
Solve for A and B: A = 4, B = -1
∫ [4/(x-3) - 1/(x+2)] dx = 4ln|x-3| - ln|x+2| + C

3. Applications of Integrals (Medium Priority)

Volume of Revolution

Disk Method: V = π ∫ [R(x)]² dx
Washer Method: V = π ∫ ([R(x)]² - [r(x)]²) dx
Shell Method: V = 2π ∫ (radius)(height) dx

Surface Area

SA = 2π ∫ y √(1 + (dy/dx)²) dx (revolution about x-axis)

Arc Length

L = ∫ √(1 + (dy/dx)²) dx

Problem Solving Strategy:

1. Identify the type of application (volume, surface area, arc length)
2. Choose the appropriate formula
3. Set up the integral with correct limits
4. Simplify the integrand before integrating
5. Evaluate the definite integral

4. Differential Equations (Medium Priority)

Separable Equations

dy/dx = g(x)h(y) → ∫ dy/h(y) = ∫ g(x) dx

First Order Linear

dy/dx + P(x)y = Q(x)
Integrating Factor: I(x) = e^(∫P(x)dx)
Solution: y = [∫I(x)Q(x)dx + C] / I(x)

Example: xy' - 2y = x², x > 0

Rewrite: y' - (2/x)y = x
Integrating factor: I(x) = e^(∫-2/x dx) = e^(-2ln|x|) = 1/x²
Multiply: (1/x²)y' - (2/x³)y = 1/x
Left side is d/dx[y/x²], so integrate both sides
Final solution: y = x²(ln|x| + C)
