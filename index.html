Part 1: Theorems with Proofs and Memory Devices

1. Extreme Value Theorem

Statement: If f is continuous on [a,b], then f attains absolute max and min values on [a,b].

Proof Sketch:

1. Continuity on closed interval ‚Üí function is bounded
2. Let M = sup{f(x) : x ‚àà [a,b]} and m = inf{f(x) : x ‚àà [a,b]}
3. By completeness of ‚Ñù, ‚àÉ c,d ‚àà [a,b] such that f(c) = M, f(d) = m
Memory Device: "Closed + Continuous = Guaranteed Extremes"

2. Rolle's Theorem

Statement: If f continuous on [a,b], differentiable on (a,b), and f(a) = f(b), then ‚àÉ c ‚àà (a,b) with f'(c) = 0.

Proof Sketch:

1. If f constant ‚Üí f'(x) = 0 everywhere
2. If f not constant ‚Üí by EVT has max/min in (a,b)
3. By Fermat's Theorem, f'(c) = 0 at that point
Memory Device: "Same height at ends ‚Üí somewhere flat in between"

3. Mean Value Theorem (MVT)

Statement: If f continuous on [a,b] and differentiable on (a,b), then ‚àÉ c ‚àà (a,b) such that:

$$f'(c) = \frac{f(b) - f(a)}{b - a}$$

Proof Sketch:

1. Define helper function: \(g(x) = f(x) - f(a) - \frac{f(b)-f(a)}{b-a}(x-a)\)
2. Check g(a) = g(b) = 0
3. Apply Rolle's Theorem to g(x)
4. g'(c) = 0 ‚Üí \(f'(c) = \frac{f(b)-f(a)}{b-a}\)
Memory Device: "Some instantaneous rate equals average rate"

4. Fundamental Theorem of Calculus

Part 1: \(\frac{d}{dx}\left[\int_{a}^{x} f(t)dt\right] = f(x)\)

Proof Sketch:

1. Let \(F(x) = \int_{a}^{x} f(t)dt\)
2. \(\frac{F(x+h) - F(x)}{h} \approx \frac{1}{h}\int_{x}^{x+h} f(t)dt \approx f(x)\)
3. Limit h‚Üí0 gives F'(x) = f(x)

Part 2: \(\int_{a}^{b} f(x)dx = F(b) - F(a)\) where F' = f

Proof Sketch:

1. Let \(F(x) = \int_{a}^{x} f(t)dt + C\)
2. F(b) - F(a) = \(\int_{a}^{b} f(t)dt\)
Memory Device: "Derivative undoes integral, integral undoes derivative"

Fermat's Theorem - Complete Proof

Fermat's Theorem (Critical Point Theorem)

Statement: If a function f has a local maximum or minimum at point c, and if f'(c) exists, then f'(c) = 0.

Complete Proof in Words:

Let's assume f has a local maximum at c (the proof for a local minimum is similar). By definition of local maximum, there exists some small interval around c where f(x) ‚â§ f(c) for all x in that interval.
Now consider the derivative from the right: f'(c‚Å∫) = lim[h‚Üí0‚Å∫] [f(c+h) - f(c)]/h Since f(c+h) ‚â§ f(c) for small positive h, the numerator f(c+h) - f(c) is ‚â§ 0, while the denominator h is positive. Therefore, the difference quotient [f(c+h) - f(c)]/h ‚â§ 0. Taking the limit as h approaches 0 from the right, we get f'(c‚Å∫) ‚â§ 0.
Now consider the derivative from the left: f'(c‚Åª) = lim[h‚Üí0‚Åª] [f(c+h) - f(c)]/h For small negative h, we still have f(c+h) ‚â§ f(c), so the numerator f(c+h) - f(c) is ‚â§ 0, but now the denominator h is negative. A negative number divided by a negative number gives a positive result, so [f(c+h) - f(c)]/h ‚â• 0. Taking the limit as h approaches 0 from the left, we get f'(c‚Åª) ‚â• 0.
Since we assumed f'(c) exists, the left-hand and right-hand derivatives must be equal: f'(c‚Å∫) = f'(c‚Åª) = f'(c). But we have f'(c) ‚â§ 0 from the right-side analysis and f'(c) ‚â• 0 from the left-side analysis. The only number that is simultaneously ‚â§ 0 and ‚â• 0 is 0. Therefore, f'(c) = 0.
Memory Aid: "At the top of a hill or bottom of a valley, the ground is flat."

Mean Value Theorem for Integrals - Complete Proof

Mean Value Theorem for Integrals

Statement: If f is continuous on [a,b], then there exists some c in [a,b] such that:

$$f(c) = \frac{1}{b-a} \int_{a}^{b} f(x)dx$$

Complete Proof in Words:

Since f is continuous on the closed interval [a,b], by the Extreme Value Theorem, f attains both an absolute minimum value m and an absolute maximum value M on [a,b]. That is, m ‚â§ f(x) ‚â§ M for all x in [a,b].
Now, using basic properties of definite integrals, we can "sandwich" our integral between these bounds: $$\int_{a}^{b} m dx \leq \int_{a}^{b} f(x)dx \leq \int_{a}^{b} M dx$$
The integral of a constant is just that constant times the length of the interval: $$m(b-a) \leq \int_{a}^{b} f(x)dx \leq M(b-a)$$
Dividing all parts of this inequality by (b-a) gives: $$m \leq \frac{1}{b-a} \int_{a}^{b} f(x)dx \leq M$$ So the number \(\frac{1}{b-a} \int_{a}^{b} f(x)dx\) lies between the minimum and maximum values of f on [a,b].
Now, since f is continuous on [a,b], by the Intermediate Value Theorem, f must take on every value between its minimum m and maximum M. In particular, there must be some point c in [a,b] where f(c) exactly equals this intermediate value: $$f(c) = \frac{1}{b-a} \int_{a}^{b} f(x)dx$$ This completes the proof.
Memory Aid: "The average height equals some actual height."
